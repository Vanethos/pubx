# pubx

The missing `pub` commands.

## Installation & Usage

```sh
pub global activate pubx
pubx search json
```

## Commands

### search

*Aliases: se, s, find*

Searches [pub.dev](https://pub.dev).

`pubx search {query}`

### view

*Aliases: info, show, v*

Displays information about the specified package.

`pubx view {package-name}`
